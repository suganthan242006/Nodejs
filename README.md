# Student & Staff Management API

A backend API built using Node.js, Express.js, TypeScript, and MongoDB to manage student and staff data.

## Features
- Student CRUD operations
- Staff CRUD operations
- JWT Authentication
- Role-based Authorization
- Input Validation

## Tech Stack
- Node.js
- Express.js
- TypeScript
- MongoDB
- JWT

## How to Run
1. Clone the repository
2. Run `npm install`
3. Create a `.env` file
4. Run `npm run dev`
